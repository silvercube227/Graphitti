Detailed Description

A class that performs stimulus input (implementation Regular).

//

The SInputRegular performs providing stimulus input to the network for each time step. Inputs are series of current pulses, which are characterized by a duration, an interval and input values.

This class is the base class of GpuSInputRegular and HostSInputRegular.
